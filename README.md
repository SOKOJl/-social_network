# -social_network
Движок социальной сети.
Использованные технологии:
PHP, SQL, HTML, CSS, JS.
Реализовано:
- Авторизация
- Администрирование
- Работа с базой данных MySQL
- Создание/редактирование/удаление пользователя и его персональной страницы
- Поиск пользователей по критериям
- Добавление в друзья
- Обмен сообщениями
